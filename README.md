# Flask To-Do Application – Git Assignment

## Overview
This project demonstrates Git operations and a Flask-based To-Do application with MongoDB integration.  
The assignment covers repository creation, SSH setup, branching, merging, conflict resolution, reset, and rebase.

---

## Technologies Used
- Python (Flask)
- MongoDB
- MongoDB Compass
- Git & GitHub
- Ubuntu Linux

---

## Branches Used
- main – Final stable branch
- shakir – Initial development
- shakir_new – JSON update
- master_1 – Frontend To-Do form
- master_2 – Backend API & MongoDB

---

## Application Features
- To-Do form with multiple fields
- Backend API to store data
- MongoDB database integration
- Verified using MongoDB Compass
